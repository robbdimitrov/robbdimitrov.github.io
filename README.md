# Robert Dimitrov - CV

Source code for my web-based interactive curriculum vitae.

## Features

- **Modern Architecture**: Built entirely with Semantic HTML5, Vanilla JavaScript, and Tailwind CSS.
- **Dynamic Theming**: Fully supports light and dark modes with a custom glassmorphism toggle.
- **Spotlight UI**: Implements a high-end, responsive radial gradient tracking effect linked to mouse movements.
- **Print Optimized**: Features a dedicated `@media print` stylesheet that automatically reformats the CV into a clean, traditional layout suitable for PDF generation and ATS parsers.
- **SEO Ready**: Includes Open Graph and Twitter Card meta tags for rich link previews.

## Running Locally

Since the project uses entirely vanilla web technologies without any build steps or external assets, you can view it locally simply by opening `index.html` in any modern web browser.
